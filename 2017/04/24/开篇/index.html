搭建博客的原因


　　很早之前就有自己建站搭博客的想法，可是作为一个非处女座完美主义者常常犹犹豫豫，未曾积极地尝试一下。这种完美主义倾向渐渐显露出侵害人格的一面，发展下去就是晚期的拖延症，同样强烈感觉是愈演愈烈的节奏。一面对自己要求越来越高，觉得一开始做的不完美就不愿意立刻行动；另一方面对周围的任何人事都充满了厌恶的情感。

　　突然间想到自己这个样子好可怕，感觉到这就是别人口中冥顽不灵那类人。就如鲁迅先生说的那样：

“不在沉默中爆发，就在沉默中消亡。”

无法继续忍受了，自然而然就到达了一个突变节点。我知道改变不是一瞬间的事情，好习惯养成需要的是坚持和思考，改变同样需要勇气。

　　综上，搭建此博客的原因有以下几点：

1.从现在开始立刻改变自己拖延的毛病，培养即刻实践的行动力。

2.阶段性地记录内心的感受和情绪，注意观察生活的细节，保持敏锐地情感触觉。

3.时刻记录学习的过程，“积跬步，至千里”量化成长与进步。

搭建博客的过程


　　从零开始查询个人博客搭建的方法，利用github网站的pages介绍主页，结合开源的hexo博客框架搭建。hexo的运行基于node.js之上，所以首先要去 node.js官网下载并安装好node.js。

　　然后下站安装 github的gitbash工具，此工具是之后命令操作的平台。安装好 gitbash之后需要配置本地电脑与网站 github的连接。

　　在 gitbash命令行中输入以下代码即刻：


  git config –global user.name   “github用户名”
  git config –global user.email  “github注册邮箱”

　　配置完github连接之后，还要去github个人setting页面去配置 ssh 建立本地操作电脑与github站点的安全链接。

　　接下来简略的介绍以下后面的搭建步骤：

1.在本地路径下建立一个blog文件夹，用于下载 hexo，进入blog路径下执行


  cd f:blog
  npm install hexo-cli g
  hexo init blog                “初始化博客”
  hexo g                        “生成博客文件”
  hexo s                        “本地部署hexo博客 通过localhost：4000端口查看”
  ctrl+c
  hexo d                        “向github上部署”

这样一个博客系统初步搭建完成此时博客地址就是https://user.github.io

2.下载next博客主题

进入blog根目录下执行代码


  git clone https://github.com/iissnan/hexo-theme-next themes/next

然后就可以在自己的blog路径的themes文件夹下看到一个next文件夹了。

3.修改主目录下的配置文件_config.yml把里面的主题theme:landscape修改为theme:next即可。重新部署博客文件，执行代码：


  hexo g    “生成博客文件”
  hexo s    “本地模拟运行localhost：4000查看效果”
  hexo d    “部署到github上”

4.此后有在万网上购买了一级域名绑定到github上，实现了一级域名化方便登陆，提升逼格。

综上


　　建站过程大致就如上所述，其中一些细节部分都可以在网上查看到，以后想到了会继续更新添加。至此，建好个人博客，开启一个新的篇章。铭记 “ 靡不有初，鲜克有终。”
